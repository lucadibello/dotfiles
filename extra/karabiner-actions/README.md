# My karabiner-elements actions

- `caps_lock.json`: remaps Caps Lock to Escape for easier modal switching.
- `right_command_shift_cmd_space.json`: remaps right Command to `Shift+Command+Space` for triggering Shortcat.
- `right_option_tmux_prefix.json`: remaps right Option to the tmux prefix `Control+B` for quick access to tmux commands.
